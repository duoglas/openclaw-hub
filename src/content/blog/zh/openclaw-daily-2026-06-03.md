《AI、科技日报》  
2026-06-03 早报

## 今日要闻（5条）

1. NVIDIA 与 Microsoft 把“智能体 AI”推向 Windows、Azure 和本地部署

发生了什么：  
NVIDIA 披露，双方在 Microsoft Build / GTC Taipei 上扩展合作：RTX Spark、DGX Station for Windows、Microsoft Fabric GPU 加速、Foundry 上的 NVIDIA 开放模型、GitHub Copilot 中的 NVIDIA OpenShell 安全运行时等。

为什么重要：  
这不是单点模型更新，而是把 AI Agent 的运行栈做成“端侧 PC + 云 + 本地/私有部署”的完整链路。

可能影响：  
开发者以后更容易在 Windows 本机跑 Agent；企业会更关注“本地数据 + 云端算力 + 安全沙箱”的混合部署。

状态：已确认（L1）


2. NVIDIA 推出 NemoClaw 路线，工业软件开始做“自主 AI 工程师”

发生了什么：  
NVIDIA 称，Cadence、Dassault Systèmes、Siemens、Synopsys 等工业软件厂商正在用 NVIDIA NemoClaw / OpenShell 构建面向设计、仿真、EDA、制造流程的长任务 AI Agent。

为什么重要：  
AI Agent 正从写文档、写代码，进入工业设计和仿真流程。重点不只是“聊天”，而是自动完成 CAD、网格划分、仿真设置、调试、报告生成等链路。

可能影响：  
工程软件会进入“Agent 化”阶段；工业 AI 的门槛会从模型能力转向安全运行时、工具权限和工作流集成。

状态：已确认（L1）


3. NVIDIA AI Cloud 生态扩张，AI 基建继续向“区域化/主权化”推进

发生了什么：  
NVIDIA 称，其 AI Cloud 生态覆盖继续扩大，新增或扩展多个区域伙伴，服务训练、微调、推理、Agentic AI、Physical AI 和 Sovereign AI。

为什么重要：  
AI 竞争正在从“谁有模型”转向“谁有低成本、低延迟、合规的 AI 工厂”。

可能影响：  
各国、云厂商、运营商会继续建设区域 AI 算力；企业部署 AI 时会更关注数据所在地、合规和 token 成本。

状态：已确认（L1）


4. OpenAI 给 ChatGPT 增加“活跃会话管理”

发生了什么：  
OpenAI 6月2日发布说明：ChatGPT 推出 Active sessions，用户可在 Settings > Security > Active sessions 查看账号会话，并登出单个或全部不认识的会话。

为什么重要：  
AI 账号正在承载文件、代码、API、工作流和个人数据。账号安全开始变成 AI 产品的核心功能，而不是附属设置。

可能影响：  
普通用户应定期检查 ChatGPT 登录设备；企业用户也会更重视 AI 工具的会话、权限、审计能力。

状态：已确认（L1）


5. 国家数据局强调：高质量数据集是具身智能、AI for Science 的基础

发生了什么：  
新华社报道，国家数据局局长刘烈宏在 2026 世界智能产业博览会上表示，高质量数据集是具身智能“感知-决策-执行”的重要基础，要以完善的数据工程驱动具身智能发展。

为什么重要：  
中国 AI 产业政策重点继续从“大模型”扩展到“数据集、行业场景、具身智能、AI for Science”。

可能影响：  
工业制造、交通、文旅、科研等行业数据集建设会加速；数据治理、标注、合成数据、隐私保护平台可能成为受益方向。

状态：已确认（L2）


## 实战案例（1-2个）

1. 普通用户：今天就检查一次 ChatGPT 活跃会话

怎么做：  
进入 ChatGPT 设置 → Security → Active sessions。  
看到陌生设备、陌生地点、旧设备，就单独登出或全部登出。

价值：  
AI 账号里可能有文件、代码、聊天记录、API 平台会话。这个动作比“换复杂密码”更直接，能快速降低被盗用风险。


2. 产业案例：宇树科技 IPO 过会，具身智能进入资本市场验证期

发生了什么：  
新华社报道，宇树科技科创板 IPO 申请已通过上交所上市审核委员会审议。公司聚焦人形机器人、四足机器人、机器人组件及具身智能模型。

为什么重要：  
这说明具身智能不再只是实验室概念，开始接受收入、利润、产能、研发投入和资本市场的硬检验。

可能影响：  
机器人本体、执行器、传感器、控制系统、具身智能模型与行业数据集会继续获得资本和产业关注，但收入质量、交付能力和安全合规会成为更硬的验证标准。

## 今日结论

- 最值得关注：企业级 AI 正在加速进入核心业务流程，AI 不再只是聊天工具，而是在进入税务、法务、制造、运维、推理服务等真实生产系统。
- 给普通用户建议：短期优先选择权限透明、可断开授权、有来源标注的 AI 产品，把它用于信息整理、学习复盘、日常文档和低风险决策辅助。
- 给团队建议：不要只比较模型榜单，先选一个高频流程做试点，并把权限、审计、成本和人工复核写进上线标准。
